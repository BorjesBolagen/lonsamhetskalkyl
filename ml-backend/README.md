# ML backend (placeholder)

This folder contains a minimal placeholder service for future ML and heavy calculations. The current app runs fullstack inside Next.js and does not depend on this service.

## Run locally

```bash
npm install
npm run dev
```

Default server URL: http://localhost:5050

## Routes

- `GET /health` - Health check
- `POST /api/ml` - Placeholder for future ML calculations (returns 501)
