## supabaseServerSchema:

This file was auto generated by Supabase CLI. Do not edit manually.
It contains the schema of the database on supabase, so we can use it here cause apparently we need it ig.
So if we change database schema in supabase, we should regenerate this file

Regenerate in terminal (from the web folder):

### Variant A 

1. npx supabase login
2. npx supabase gen types typescript --project-id ukfnyyglaistpbnlgjar > lib/supabaseServerSchema.ts

### Variant B (recommended for PowerShell)

1. npx supabase login
2. npx supabase gen types typescript --project-id ukfnyyglaistpbnlgjar | Out-File -FilePath lib/supabaseServerSchema.ts -Encoding utf8

PowerShell + UTF-8 note:
Using > in PowerShell can create a file with a problematic encoding in some setups.
Use Out-File with -Encoding utf8 to avoid encoding issues (for example Next.js/Turbopack failing to parse the file).
